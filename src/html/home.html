Results Directory:

Overview

RoundTabler is a software security analysis tool that used for scanning relational database column contents for unencrytped card and ACH information. With the ability to scan your entire database, or just one table, RoundTabler offers flexibility to all security teams depending on their needs.

Features:

  • Easy scanning of columns
  • Low false-positives
  • Avoids bad outcome (we do not certify databases with unencrypted data)
  • Reported data includes confidence scores
  • Quick Deployment Option
  • Deployable via a docker

List of parameters:

Required Parameters:

  • --type=
  • Specifies the type of scan you would like to perform. Options include: pci, nacha, or all
  • --dbtype=
  • Specifies the type of database RoundTabler will be scanning. **Currently RoundTabler only supports MariaDB**
  • --server=
  • Specifies the server address of the database
  • --user=
  • Your username in order to connect to the database server
  • --password=
  • Your password in order to connect to the database server
  • --database=
  • Specifies the name of the database you would like to scan at the server address

    • Optional Parameters:

  • --table=
  • Specifies a specific table name to be scanned instead of scanning the entire database at the server address
  • --help
  • Prints out a list of commands on your CLI for your viewing.
